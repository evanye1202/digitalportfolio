Code Translator

There are many different coding languages in the world, each with different syntax and speed. Modern languages are more complex but more efficient than than older languages, and as a result, developers often have to use older languages and losetime. To solve this, our mission is to make modern languages easier to use with a code translator. Our product will create an app that converts slower and easier languages to faster and harder languages, which can be time consuming by hand. The code converter will use a file type or txt file of the code and convert the syntax from one language to another. For our first languages, we will be trying to convert Swift to C++.
